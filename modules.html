Const sound

sound: EventEmitter<string | symbol> = ...
  • Defined in src/animate/sound.ts:13
description

Event emitter for all sound events. This emits a single play event which contains the alias, loop and MovieClip which is playing the sound.

example

PIXI.animate.sound.on('play', (alias, loop, context) => { // custom handle sounds being played // where 'alias' is the ID in stage assets });
